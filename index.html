公司概况

群丰精密电子有限公司（Q-PHONE）成立于 2005 年，是一家专注于高频电感、工频电感研发与制造的国家级高新技术企业，也是国内多基地布局的精密电子元器件核心企业。

公司总部设立于江苏省昆山市，经过二十余年的稳健发展，现已形成三大自有自建生产基地的产业布局：

  • 昆山基地：专注高频电感研发与设计，为产品创新提供核心技术支持；
  • 安徽基地：安徽省高新技术企业，高频与工频电感规模化生产基地，具备自动化产线与规模化交付能力；

公司主营 PFC 电感、共模电感、电抗器、变压器等核心产品，广泛覆盖智能家电、工业自动化、光伏储能、新能源汽车、通讯设备等多个应用领域。凭借稳定的产品品质与持续的技术创新，公司已通过 ISO9001 质量管理体系与 ISO14001 环境管理体系认证，全系列产品符合 ROHS、REACH 环保标准，并配备盐雾试验、高低温实验、材料检测、产品寿命测试等专业实验室，全方位保障产品的可靠性与安全性。

截至 2025 年，公司年销售额突破 8.7 亿元，凭借多基地协同的生产优势、完善的供应链管理体系与专业的研发能力，成为行业内极具影响力的磁性器件供应商。未来，群丰精密将继续深耕精密电子领域，坚持 “精密·品质·创新·致远” 的经营理念，为客户提供更优质的产品与服务，致力于成为行业领先的磁性器件提供商。

主营产品

共模电感

广泛应用于空调、变频家电、开关电源、UPS、新能源设备，抑制电磁干扰。

产品特点：

  • 立式/卧式/三相/纳米晶/大电流可选
  • 高共模干扰抑制能力
  • 低漏感、长寿命
  • 符合RoHS，支持定制
额定电流 0 ~ 100A
电感范围 0 ~ 100mH (Customizable)
工作温度 -40℃ ~ +125℃

PFC电感

主要用于家用/商用空调、变频电源、工业电源、新能源储能。

产品特点：

  • 扁线/圆线工艺可选
  • 空调专用、大功率适配
  • 高效率、低损耗、低噪音
  • 大电流规格齐全
额定电流 0 ~ 200A
电感范围 0 ~ 1mH (Customizable)
工作温度 -40℃ ~ +125℃

功率电感

应用于消费电子、适配器、LED驱动、控制板、小家电。

产品特点：

  • 批量一致性好
额定电流 0 ~ 30A
电感范围 0.1μH ~ 1000μH
工作温度 -40℃ ~ +125℃
