主营产品

共模电感

广泛应用于空调、变频家电、开关电源、UPS、新能源设备，抑制电磁干扰。

产品特点：

  • 立式/卧式/三相/纳米晶/大电流可选
  • 高共模干扰抑制能力
  • 低漏感、长寿命
  • 符合RoHS，支持定制
额定电流 0 ~ 100A
电感范围 0 ~ 100mH (Customizable)
工作温度 -40℃ ~ +125℃

PFC电感

主要用于家用/商用空调、变频电源、工业电源、新能源储能。

产品特点：

  • 扁线/圆线工艺可选
  • 空调专用、大功率适配
  • 高效率、低损耗、低噪音
  • 大电流规格齐全
额定电流 0 ~ 200A
电感范围 0 ~ 1mH (Customizable)
工作温度 -40℃ ~ +125℃

功率电感

应用于消费电子、适配器、LED驱动、控制板、小家电。

产品特点：

  • 贴片/直插两种封装

  • 批量一致性好

电感范围 0.1μH ~ 1000μH
工作温度 -40℃ ~ +125℃
